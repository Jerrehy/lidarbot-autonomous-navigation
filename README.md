## 📖 Описание проекта

Данный проект представляет собой полнофункциональную систему автономной навигации для мобильного робота с дифференциальным приводом. Система реализует:

- ✅ **SLAM** (Simultaneous Localization and Mapping) — построение карты неизвестной среды
- ✅ **Локализацию** с использованием AMCL и Extended Kalman Filter
- ✅ **Планирование пути** с динамическим избеганием препятствий
- ✅ **Автономное исследование** пространства (Frontier-based exploration)
- ✅ **Навигацию по заданным точкам** (Waypoint navigation)

### Ключевые особенности

- **Сенсорная фузия**: объединение данных лидара RPLIDAR A1M8 и IMU (Waveshare 10 DOF)
- **Реальное время**: все вычисления выполняются на бортовом компьютере Raspberry Pi 4
- **Модульная архитектура**: легко расширяемая система на базе ROS 2 Humble
- **Безопасность**: встроенный watchdog и аварийная остановка

## 🔧 Аппаратное обеспечение

| Компонент | Модель | Назначение |
|-----------|--------|------------|
| **Одноплатный компьютер** | Raspberry Pi 4 Model B (4GB) | Бортовой компьютер |
| **Лидар** | RPLIDAR A1M8 | Сканирование окружающей среды (360°, до 12м) |
| **IMU** | Waveshare 10 DOF (LSM6DSOX + LIS3MDL + BMP280) | Ориентация, ускорение, магнитное поле |
| **Привод** | 4 DC мотора + Motor Driver HAT | Дифференциальный привод |
| **Питание** | Li-Po аккумулятор 3S 5000mAh | Автономная работа до 2 часов |

### Стек технологий
